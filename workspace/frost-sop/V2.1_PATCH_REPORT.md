## 二、代码变更详情

### P0-1: DB 表结构修复 (`core/db.py`)

```diff
- energy_log: 仅定义基础列，migration 补加 energy_level/health_score
+ energy_log: CREATE TABLE 直接包含 energy_level/health_score

- schedule: 仅定义基础列，migration 补加 sop_id/sop_params/priority
+ schedule: CREATE TABLE 直接包含 sop_id/sop_params/priority

- decision_points: 仅定义基础列，migration 补加 stage_id/user_decision/user_note/responded_at/decision_type
+ decision_points: CREATE TABLE 直接包含所有列
```

### P0-2: 决策管理器修复 (`core/decision_manager.py`)

- 移除 `resume_decision()` 中对 `tasks.decision_status` 的不存在列更新
- 新增 `reject_decision()` 方法（显式拒绝操作）
- 添加 `threading.Lock` 线程安全保护

### P1-6: TASK_DECOMPOSED 事件 (`main.py`)

```python
# SOP 内化完成后发布 TASK_DECOMPOSED 事件
bus.publish(Event(
    event_type=EventType.TASK_DECOMPOSED,
    source="main",
    data={"task_id": task_id, "stage_count": N, "stages": [...]},
))
```

### P1-7: 敏感数据过滤 (`core/event_bus.py`)

- 新增 `_SENSITIVE_KEYS` 集合（9 个键名）
- 新增 `_sanitize_data()` 递归过滤方法
- `_persist_event()` 写入前自动调用过滤

### P1-8: 循环事件防护 (`core/event_bus.py`)

```python
if (hasattr(callback, '__name__') and callback.__name__ != '<lambda>'
        and callback.__name__ == event.source):
    continue  # 跳过同名回调防止循环
```

### P1-9: agents 表 UPSERT (`core/agent.py`)

```python
# INSERT OR REPLACE 替代 INSERT，重复运行不报 UNIQUE 约束错误
conn.execute(
    "INSERT OR REPLACE INTO agents (id, name, agent_type, generation, created_at) "
    "VALUES (?, ?, ?, ?, ?)", ...
)
```
