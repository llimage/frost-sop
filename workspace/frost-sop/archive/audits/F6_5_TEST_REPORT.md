## 一、功能概述

**F6.5 撒豆成兵** 是让系统从"技术验证品"变成"可保存、可唤醒的个人助手"的关键一步。

**核心功能**：
1. 保存当前系统配置（任务描述、项目名等）到 `assets/` 目录的 JSON 文件
2. 唤醒上次保存的配置，自动填充到界面
3. 启动时自动加载最新配置（自动唤醒）

---

## 二、实施内容

### 2.1 新增 `AssetStore` 类（`core/store.py`）

实现了以下方法：
- `save(config, filename)` - 保存配置到 JSON 文件
- `load(filename)` - 加载指定配置文件
- `list()` - 列出所有配置文件（最新的在最前）
- `load_latest()` - 加载最新的配置文件

### 2.2 修改 `app.py`

新增功能：
1. 导入 `AssetStore` 类
2. 会话状态初始化（`saved_config`, `config_loaded`）
3. 自动唤醒逻辑（在 `init_family()` 后调用 `auto_wake()`）
4. 指挥面板新增"💾 保存当前配置"和"🔄 唤醒上次配置"两个按钮
